## The Project

VertexNova is a **modern, multi-backend C++17/20 rendering engine** — built from the ground up to run on every major graphics API and platform from a single codebase. It provides a clean abstraction layer (`vnecrossgl`, `vnecrosswin`, `vnecrossviz`) over Vulkan, Metal, OpenGL, and OpenGL ES, with strict ownership, predictable lifetimes, and thin abstractions.

For platform coverage, architecture, and roadmap, see [Overview](/docs/docs/overview) and [Roadmap](/docs/docs/roadmap).
