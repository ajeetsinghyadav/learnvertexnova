# Start Here

**One codebase. Four graphics APIs. Six platforms.** VertexNova is a C++20 rendering engine that runs on Vulkan, Metal, OpenGL, and WebGL — from a single shader source. No magic, no black boxes. Just clean, readable code that teaches you how it works.

## Run something in 5 minutes

**1. Clone**

```bash
git clone --recursive https://github.com/vertexnova/vnetestbed.git
```

**2. Build**

```bash
cd vnetestbed
./scripts/build_macos.sh -t Debug -b Metal   # or build_linux.sh, build_windows.sh
```

**3. Run**

```bash
cd build/Debug/build-*/bin
./04_DemoCube
```

A window opens. A cube renders. You just ran your first demo.

:::tip Full build guide
Platform-specific scripts, iOS, visionOS, Web — see [Build & Run](/docs/docs/demos/build).
:::

## What makes this different?

- **Write once, run everywhere** — Same C++ and GLSL across Metal, Vulkan, OpenGL, Web
- **Built to learn** — Transparent architecture. Every layer is readable and documented
- **22+ demos** — Triangle → PBR → deferred → compute. Each proves one concept
