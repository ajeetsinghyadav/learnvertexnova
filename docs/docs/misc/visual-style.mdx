# VertexNova Visual Style

Consistent colors (and later, typography) for **presentations (PPT/Keynote), diagrams, and VertexNova-related design** so the project looks coherent across the site and decks. VertexNova is a project with a chosen visual style; this page documents it.

Every color has a clear RGB definition you can use in design tools and code comments.
