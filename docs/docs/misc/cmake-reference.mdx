# CMake Build System

Reference and notes for CMake тАФ the cross-platform build system used by VertexNova and most C/C++ projects.

## Overview

CMake is a **meta-build system**. It reads `CMakeLists.txt` and generates native build files (Makefiles, Ninja, Visual Studio, Xcode). You then run the native tool to compile.

| Step | Command | Does |
|------|---------|------|
| Configure | `cmake -B build` | Generate build files |
| Build | `cmake --build build` | Compile the project |
| Install | `cmake --install build` | Copy artifacts to prefix |

:::tip Two-stage process
Always keep configure and build separate. Never run `cmake .` in source dir тАФ always `cmake -B build` (out-of-source).
:::

---

## Core Concepts

- **Target** тАФ Executable or library. Carries its own compile flags, includes, and link deps.
- **CMakeLists.txt** тАФ Build definition. Every directory can have one. Root is entry point.
- **Generator** тАФ What CMake outputs: Ninja, Makefiles, Visual Studio, Xcode. Set with `-G`.
- **Build tree** тАФ Out-of-source dir (e.g. `build/`). Never build in-source.
- **Cache** тАФ `CMakeCache.txt` stores config. Override with `-D` flags.

## Project Structure

```
my-project/
тФЬтФАтФА CMakeLists.txt          # project() + subdirs
тФЬтФАтФА src/
тФВ   тФЬтФАтФА CMakeLists.txt      # add_executable / add_library
тФВ   тФФтФАтФА main.cpp
тФЬтФАтФА include/
тФВ   тФФтФАтФА renderer.h
тФЬтФАтФА libs/core/
тФВ   тФФтФАтФА CMakeLists.txt      # library target
тФФтФАтФА build/                  # OUT-OF-SOURCE тАФ git-ignored
```

### Minimal root CMakeLists.txt

```cmake
cmake_minimum_required(VERSION 3.20)
project(MyProject VERSION 1.0.0 LANGUAGES CXX)

set(CMAKE_CXX_STANDARD 17)
set(CMAKE_CXX_STANDARD_REQUIRED ON)

add_subdirectory(src)
add_subdirectory(libs/core)
```

## Targets & Linking

### Creating targets

```cmake
add_executable(myapp src/main.cpp src/app.cpp)

add_library(mylib STATIC src/renderer.cpp)    # static .a / .lib
add_library(mylib SHARED src/plugin.cpp)      # shared .so / .dll
add_library(mylib INTERFACE)                  # header-only, no sources
```

### target_* тАФ the modern way

```cmake
target_include_directories(mylib
  PUBLIC  ${CMAKE_CURRENT_SOURCE_DIR}/include   # consumers get it too
  PRIVATE ${CMAKE_CURRENT_SOURCE_DIR}/src       # only this target
)

target_link_libraries(myapp
  PRIVATE   mylib      # link, don't expose to consumers
  PUBLIC    vn_math    # link AND expose (transitive)
)

target_compile_options(mylib PRIVATE -Wall -Wextra)

target_compile_definitions(mylib PRIVATE VN_VERSION=1)
```

### PRIVATE / PUBLIC / INTERFACE

| Keyword | This target | Consumers (linking targets) |
|---------|-------------|------------------------------|
| PRIVATE | тЬУ applied | тЬЧ not propagated |
| PUBLIC | тЬУ applied | тЬУ propagated |
| INTERFACE | тЬЧ not applied | тЬУ propagated |

## Variables & Options

### Variables

```cmake
set(MY_SOURCES src/a.cpp src/b.cpp)
add_library(mylib ${MY_SOURCES})
```

| Variable | Meaning |
|----------|---------|
| `CMAKE_SOURCE_DIR` | Root source dir |
| `CMAKE_CURRENT_SOURCE_DIR` | Current CMakeLists.txt dir |
| `CMAKE_BINARY_DIR` | Root build dir |
| `CMAKE_BUILD_TYPE` | Debug \| Release \| RelWithDebInfo |
| `CMAKE_INSTALL_PREFIX` | Install destination |
| `PROJECT_SOURCE_DIR` | Nearest project() source dir |

### Options

```cmake
option(VN_BUILD_TESTS "Build tests" OFF)
option(VN_ENABLE_VULKAN "Enable Vulkan" ON)

if(VN_BUILD_TESTS)
  add_subdirectory(tests)
endif()
```

Override: `cmake -B build -DVN_BUILD_TESTS=ON`

### Generator expressions

Evaluated at build time. Syntax: `$<condition:value>`

```cmake
target_compile_options(myapp PRIVATE
  $<$<CONFIG:Debug>:-g -O0>
  $<$<CONFIG:Release>:-O3 -DNDEBUG>
)

target_compile_definitions(myapp PRIVATE
  $<$<PLATFORM_ID:Windows>:NOMINMAX>
)
```

## find_package & FetchContent

### find_package (system-installed)

```cmake
find_package(Vulkan REQUIRED)
find_package(OpenGL REQUIRED)
find_package(glfw3 3.3 REQUIRED)

target_link_libraries(myapp PRIVATE
  Vulkan::Vulkan    # imported target тАФ brings includes + libs
  OpenGL::GL
  glfw
)

# Optional dependency
find_package(Catch2 QUIET)
if(Catch2_FOUND)
  add_subdirectory(tests)
endif()
```

### FetchContent (download at configure)

```cmake
include(FetchContent)

FetchContent_Declare(
  glm
  GIT_REPOSITORY https://github.com/g-truc/glm.git
  GIT_TAG        0.9.9.8
)

FetchContent_MakeAvailable(glm)
target_link_libraries(myapp PRIVATE glm::glm)
```

:::info When to use which
**find_package** тАФ System-installed (Vulkan SDK, OpenGL). **FetchContent** тАФ Pin a version, header-only or small libs.
:::

## Commands

### Configure

```bash
cmake -B build
cmake -B build -G "Ninja"
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake -B build -DCMAKE_INSTALL_PREFIX=./dist
cmake -B build -DCMAKE_TOOLCHAIN_FILE=cmake/arm.cmake
cmake --preset=release
```

### Build & install

```bash
cmake --build build
cmake --build build --parallel 8
cmake --build build --target mylib
cmake --build build --config Release    # MSVC/Xcode
cmake --install build
cmake --install build --prefix ./dist
cmake --build build --target clean
```

### Generators

| Generator | Platform | Notes |
|-----------|----------|-------|
| Ninja | All | Fastest, recommended |
| Unix Makefiles | Linux/macOS | Default on Unix |
| Visual Studio 17 2022 | Windows | .sln, multi-config |
| Xcode | macOS | .xcodeproj, multi-config |

## Modern CMake Tips

:::danger Avoid legacy patterns
`include_directories()`, `link_libraries()`, `add_compile_options()` (without target_), and `CMAKE_CXX_FLAGS` manipulation. Use `target_*` always.
:::

- **target_* always** тАФ Every include, flag, define via target_* тАФ never global.
- **Out-of-source build** тАФ Always `cmake -B build`. Add `build/` to `.gitignore`.
- **Prefer Ninja** тАФ Faster than Make, works everywhere.
- **Imported targets** тАФ Link to `Vulkan::Vulkan`, not `VULKAN_LIBRARIES`.
- **cmake_minimum_required first** тАФ Very first line. Use 3.20+ for new projects.

### CMakePresets.json

```json
{
  "version": 6,
  "configurePresets": [
    {
      "name": "release",
      "generator": "Ninja",
      "binaryDir": "${sourceDir}/build/release",
      "cacheVariables": {
        "CMAKE_BUILD_TYPE": "Release",
        "VN_ENABLE_VULKAN": "ON"
      }
    }
  ]
}
```

```bash
cmake --preset=release
cmake --build --preset=release
```
