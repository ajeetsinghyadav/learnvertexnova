# Demos

VertexNova ships with **22+ working demos** spanning Metal, Vulkan, OpenGL, Web, iOS Sim, and visionOS. Each demo is a focused example of a specific rendering technique — with clean, readable source code.

:::tip Browse the source
All demos are open source. [View on GitHub →](https://github.com/ajeetsinghyadav/vertexnova)
:::

---

## Building and Running

VertexNova provides platform-specific build scripts in the `scripts/` directory. Build, then run demos from the output `bin/` folder.

### Build

<Tabs>
<TabItem value="macos" label="macOS" default>

```bash
cd vertexnova
./scripts/build_macos.sh -t Debug -b Metal
# Demos: build/Debug/build-macos-clang-<version>/bin/
```

</TabItem>
<TabItem value="ios" label="iOS">

```bash
./scripts/build_ios.sh -t Debug -simulator
```

</TabItem>
<TabItem value="visionos" label="visionOS">

```bash
./scripts/build_visionos.sh -t Debug
```

</TabItem>
<TabItem value="windows" label="Windows">

```powershell
.\scripts\build_windows.sh -t Debug -b Vulkan
# Demos: build\Debug\build-windows-cl-<version>\bin\
```

</TabItem>
<TabItem value="linux" label="Linux">

```bash
./scripts/build_linux.sh -t Debug -b Vulkan
# Demos: build/Debug/build-ubuntu-<compiler>-<version>/bin/
```

</TabItem>
<TabItem value="web" label="Web">

```bash
./scripts/build_web.sh Release
# Output: build/Release/web/
```

</TabItem>
</Tabs>

### Run demos

```bash
# From vertexnova repo root (path varies by platform and compiler version)
cd build/Debug/build-macos-clang-17.0.0/bin   # macOS example
./02_DemoTriangle
./04_DemoCube
./12_DemoLighting
```

For full setup and manual CMake builds, see the [Getting Started guide](/docs/intro).
