# Project Overview

VertexNova is a personal, part-time project focused on learning and demonstrating **cross-backend engine design**.

## 2-minute overview

- **What:** multi-backend renderer/visualization engine (Vulkan · Metal · OpenGL · Web)
- **Why:** consistent abstractions + real demos + practical learning material
- **How to evaluate:** run demos, skim architecture docs, review roadmap/status

## Best places to look

- **Getting Started:** [/docs/intro](/docs/intro)
- **Demos catalog:** [/docs/demos](/docs/demos)
- **Roadmap/status:** [/docs/roadmap](/docs/roadmap)

## Engineering highlights (suggested topics to expand)

- Resource binding + shader reflection strategy across backends
- Render targets + MSAA resolve differences (Vulkan/Metal/OpenGL)
- Debugging strategy (validation layers, markers, capture tools)
- Performance work (pipeline caching, batching/instancing, state validation)
